# Some helpful commands


### Common

Search and Grep
<leader>sg

Select buffer
<leader>fb

Go to previous buffer
```nvim
[b
```
Go to next buffer
```nvim
]b
```
Seach for keymaps
```nvim
<leader>sk
```
Toggle options
```nvim

<leader>u
```
Quick terminal

```nvim
<leader>ft
```
### LSPs

If you want to rename a function throughout your project you can use:

<leader>cr
